## 💡 Quick Reference

### News Sources
- Yahoo Finance (Stock-specific)
- MarketWatch (Market trends)
- CNBC (Breaking news)

### API Endpoints
- `GET /api/news/stock/{symbol}` - Stock news
- `GET /api/news/market` - Market news
- `GET /api/news/trending` - Trending news

### Sentiment Colors
- 🟢 Green = Positive
- 🔴 Red = Negative
- ⚪ Gray = Neutral

### File Locations
- Backend: `backend.py/news_scraper.py`
- Frontend Hook: `src/hooks/useNews.js`
- Frontend Component: `src/pages/NewsPage.jsx`
- API: `backend.py/server.py`
